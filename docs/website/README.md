# 마리코 & 유키에 공식 웹사이트 — 문서

마리코 & 유키에 (Mariko & Yukie)의 **영구 디지털 거점** 웹사이트를 위한 기획·설계 문서 모음입니다.
2026년 8월 발매되는 앨범 《남산타워》를 첫 콘텐츠로 출발하되, 장기적으로 두 아티스트의 음악·이력·아카이브를 축적하는 공식 사이트를 지향합니다.

---

## 이 사이트가 무엇인가

- **목적:** 두 아티스트의 영구 디지털 거점 (텀블벅 펀딩 유도는 보조 동선)
- **언어:** 한국어·일본어 양언어 (동등 원칙)
- **기술:** Next.js / TypeScript, 마크다운·파일 기반 콘텐츠, 정적 생성
- **개발·운영:** 황경하 (스튜디오 놀) 단독
- **제작 전제:** 텀블벅 펀딩 달성 여부와 무관하게 기본 제작 (기획안 PART 7-3)
